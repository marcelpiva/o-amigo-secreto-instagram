# Estratégia de Humanização - Posts com Fotos Reais

## Conceito

Posts humanizados com fotos de pessoas reais (royalty-free) para criar conexão emocional e autenticidade. Foco em storytelling sobre a comunidade e o propósito do app.

## Diretrizes de Seleção

### DO (Fazer)

- Escolher fotos com pessoas sorrindo genuinamente
- Priorizar diversidade étnica e de idade
- Usar fotos com iluminação quente e acolhedora
- Selecionar imagens que pareçam espontâneas
- Incluir fotos com 2+ pessoas (comunidade)
- Preferir expressões autênticas vs poses artificiais

### DON'T (Evitar)

- Fotos muito "stock" / artificiais
- Modelos obviamente posando
- Iluminação fria/corporativa
- Homogeneidade (só pessoas brancas/jovens)
- Fotos com marcas visíveis
- Imagens muito editadas/filtradas

## Composição dos Posts Humanizados

### Estrutura Recomendada

```
┌─────────────────────────────┐
│                             │
│    [FOTO HUMANIZADA]        │
│    (70% do espaço)          │
│                             │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │ Texto + Logo        │    │
│  │ (30% inferior)      │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
```

### Overlay de Texto

- Fundo semi-transparente escuro (rgba 0,0,0,0.6)
- Texto branco para contraste
- Logo do app pequeno no canto
- Fonte legível (sem serifas)

## Tom de Voz para Posts Humanizados

### Princípios

1. **Vulnerabilidade autêntica**: "Somos uma equipe pequena..."
2. **Gratidão genuína**: "Cada seguidor importa..."
3. **Propósito claro**: "Acreditamos que tecnologia deve ser gratuita..."
4. **Chamada à ação suave**: "Se você concorda, nos ajude..."

### Frases-Chave

```
"Feito por brasileiros, para brasileiros."
"Gratuito porque acreditamos que celebrar não deve custar nada."
"Cada seguidor nos ajuda a continuar."
"Sem investidores. Sem anúncios. Só comunidade."
"Você não é o produto. Você é a razão de existirmos."
```

### Evitar

- Desespero ("Por favor, sigam!")
- Mendicância ("Precisamos de seguidores!")
- Comparações negativas com concorrentes
- Promessas vazias

---

## Métricas de Sucesso

### Posts Humanizados devem ter:

- **Taxa de salvamento** > 2% (pessoas guardam conteúdo emocional)
- **Compartilhamentos** > posts promocionais
- **Comentários** com histórias pessoais
- **Conversão em seguidores** > média

### A/B Testing

Testar:
1. Foto de família vs foto de amigos
2. Texto longo vs texto curto
3. Com emoji vs sem emoji
4. CTA direto vs CTA suave
